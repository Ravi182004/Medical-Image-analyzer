About the Project

Medical Image
Analyzer

AI-driven diagnostic assistance powered by deep learning and convolutional neural networks.

The Medical Image Analyzer is an AI-powered web application designed to assist in the analysis of medical images such as MRI scans, CT scans, and Chest X-rays. The system leverages advanced Deep Learning techniques and Convolutional Neural Networks (CNNs) to detect diseases and provide preliminary diagnostic insights.

Key Features
  • ✦
    Multi-Modal Image Support: Supports MRI, CT scans, and X-ray images for comprehensive analysis.
  • ✦
    Automated Disease Detection: Identifies conditions such as Brain Tumor, Lung Cancer, Pneumonia, COVID-19, and Tuberculosis.
  • ✦
    Doctor Recommendations: Suggests relevant specialists based on detected disease.
  • ✦
    Diet Plan Guidance: Personalized diet recommendations based on disease and user preference.
  • ✦
    BMI Calculator: Helps users assess their health status and receive diet suggestions.
  • ✦
    Chatbot Support: A simple chatbot to answer common health-related questions.
  • ✦
    History Tracking: Stores previously uploaded images and results for user reference.
Technologies Used
Python (Flask) TensorFlow / Keras CNN Models HTML & CSS NumPy OpenCV
Objective
The goal of this project is to streamline medical image analysis by integrating AI-driven predictions with actionable healthcare guidance. It aims to support early detection, improve accessibility to medical insights, and assist both patients and healthcare professionals in making informed decisions.
